# proj_Node_publicAPI
공공API를 활용한 간단한 API서버를 구축함.



# 주제
버스 도착 정보 API: 대중 교통 관련 API를 활용하여 특정 정류장의 버스 도착 정보를 조회하는 API 서버.
사용자가 정류장 번호나 위치를 입력하면 해당 정류장의 버스 도착 예정 시간과 버스 노선 정보를 반환하는 API를 개발하였습니다.



# ERD 
- 로그인을 해야 해당 정류장에 도착하는 버스 정보를 조회 할수 있도록 함.
